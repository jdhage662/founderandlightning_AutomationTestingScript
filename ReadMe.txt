1. Create Empty folder where you want to create project

2. Clone the project from the git or if it is zip file download and unzip in particulr folder

3. Install and open Visual Studio Code Editor

4. Click on the File and select open as Add Folder to workspace

5. Select the clone or extracted project

6. Install node.js if it is not their in your system

7. open VS terminal 

8. type npm init

9 . npm install cypress --save-dev 

10. npm Cypress Open if you want to execute the test cases from the cypress runner

11. Run script from command line using  "npx cypress run  --spec cypress/e2e/TestCases/ContactAction.cy.js --browser chrome "
